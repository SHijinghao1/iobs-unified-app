一、界面总览
1.1 设计理念
New Space控制系统采用沉浸式3D可视化操作模式，整合手术床、C臂影像设备、AGV自动导航小车三大核心设备的控制功能。以“所见即所得”为核心，通过三维实景模拟真实手术室设备状态，替代传统纯参数操控方式，操作直观、上手简单，适配各类手术场景使用。
1.2 整体布局结构
界面采用五分区对称布局，分区清晰、功能独立，互不干扰，全方位覆盖术前、术中、术后全流程操作：
1. 顶部导航栏：功能切换、设备状态监测、系统时间展示
2. 左侧控制面板：核心设备手动调控、参数设置、体位管理
3. 中央3D场景区：设备三维实景展示、视角操控、实时状态同步
4. 右侧辅助边栏：手术进度管控、场景预设、辅助功能配置
5. 底部功能栏：全局安全操作、系统工具、远程协作功能


1.3 核心功能模块汇总
功能模块	核心功能定位	日常主要操作
手术床控制	手术床全关节状态监测与姿态调节	升降、平移、倾斜、折角调节，运动速度设置
C臂影像控制	影像设备姿态调控与设备联动管理	手动调位、联动模式切换、设备姿态重置
AGV小车控制	自动导航小车运动控制与床机对接	方向移动、升降调节、自动寻位、速度调控
手术体位预设	标准化手术体位快速调用与自定义保存	一键应用标准体位、保存专属体位、编辑体位参数
束光器可视化	影像手术光束范围精准调节与预览	光束大小、角度调整，照射区域预览演示
3D场景交互	设备全景可视化查看与视角管控	视角旋转、缩放、平移、视角锁定、设备高亮定位

二、顶部导航栏功能说明

2.1 区域样式与基础属性
顶部导航栏为界面固定常驻区域，采用半透明磨砂质感设计，底部细分隔线区分主体区域，整体简洁清晰，无遮挡、无视觉干扰，全程常驻界面顶部，方便随时查看状态、切换功能。
2.2 三大功能区域详解
2.2.1 左侧品牌与模块区域
展示系统品牌名称及当前操作模块，实时提示操作人员所处功能页面，避免操作混淆，确保操作精准性。

2.2.2 中间功能导航按钮区域
为系统核心功能切换入口，点击即可快速跳转对应操作页面，按钮状态实时区分激活与未激活状态，辨识度极高。
功能按钮	使用状态	功能说明
手术床控制	默认激活	进入手术床核心控制页面，进行全关节手动调节
体位管理	可切换激活	管理各类标准手术体位，适配不同手术场景需求
记忆体位	可切换激活	查看、调用、编辑已自定义保存的专属手术体位
设备联动	可切换激活	配置手术床、C臂、AGV小车多设备协同运行规则
设置	可切换激活	进入系统全局配置页面，调整设备与界面基础参数
按钮状态规则：当前使用页面按钮高亮显示，未使用页面按钮为常规灰色，鼠标悬浮自动提亮，点击切换流畅无卡顿。

2.2.3 右侧状态与时间区域
实时展示系统运行状态、设备连接状态及系统时间，是设备运行状态的核心监测窗口。
状态图标	状态说明
蓝牙图标	常亮正常颜色，代表蓝牙设备正常连接、可正常通信
网络图标	绿色=后端设备通信正常；红色=网络断开、设备断连，需及时排查
信号图标	固定显示，直观展示整体设备信号强弱状态
系统时间	实时刷新展示当前时间、日期，可用于手术操作时间记录溯源

2.3 基础使用规范
1. 操作前优先查看右侧网络状态，红色断连时禁止进行设备调节操作；
2. 通过顶部按钮精准切换功能模块，避免跨页面误操作；
3. 以界面实时时间为准，记录手术关键操作节点。
三、左侧控制面板核心操作
3.1 面板基础属性
左侧控制面板为系统核心操作区域，固定悬浮于界面左侧，尺寸适配全屏界面，采用磨砂半透明设计，不遮挡3D场景，支持四种功能视图自由切换，涵盖所有设备手动控制功能。
3.2 四大视图切换规则
视图模式	进入方式	核心用途
手术床仪表板（默认）	系统默认首页	手术床10个关节实时监控、手动调节、速度设置
C臂控制面板	3D场景C臂卡片点击更多	C臂影像设备姿态调节、联动模式配置、参数重置
AGV小车控制面板	3D场景AGV卡片点击更多	导航小车移动控制、自动寻位、速度调节
姿态预设库	首页姿态预设点击更多	标准体位调用、自定义体位保存与管理
3.3 手术床仪表板操作（核心常用）
3.3.1 关节功能说明
手术床共设置10个可独立调控关节，覆盖床体升降、平移、倾斜、分段折角等全部姿态调节功能，每个关节独立显示实时数值、运行进度，支持精准微调。
3.3.2 基础运动操作流程
启动运动：找到对应关节，点击方向控制按钮，按钮激活高亮，设备开始运行，界面弹出操作成功提示。
停止运动：再次点击当前激活的方向按钮，设备立即停止运行，按钮恢复默认状态。
关节选中定位：点击任意关节条目，条目高亮，同时3D场景中对应床体部件同步高亮，方便精准核对设备姿态；再次点击即可取消选中。
3.3.3 速度设置功能
1. 点击页面右上角「速度设置」按钮，进入调速模式；
2. 可通过滑块拖动或直接输入数值，修改单个关节的运行速度；
3. 调节完成后自动生效，如需恢复默认参数，点击「重置默认速度」即可一键还原。
3.4 C臂影像设备控制面板
3.4.1 联动模式设置
C臂设备可与手术床实现智能联动，适配不同手术操作需求，共三种联动模式，可自由切换：
同步模式：C臂跟随手术床同步运动，适用于常规手术，保证视野稳定跟随。
镜像模式：C臂与手术床反向运动，适配特殊手术视角需求。
脱离模式：C臂独立运行，不受手术床姿态影响，可单独调节影像位置。
同时支持自动跟随、手动控制两种操作模式，可根据手术场景灵活选择。
3.4.2 手动关节调节
支持立柱升降、前后移动、斜臂旋转、C环滚动四项核心调节，可通过滑块精准微调或方向按钮快速调控。点击页面底部「重置本地关节位姿」，可一键将C臂恢复至初始标准姿态，清空所有自定义参数。
3.5 AGV小车控制面板
3.5.1 设备状态监测
页面顶部实时展示AGV小车核心状态，包含设备运行状态（就绪/寻位中/已停止）、剩余电量、信号强度，方便操作人员实时掌握设备工况。
3.5.2 运动控制操作
1. 方向控制：支持前进、后退、左移、右移、左旋、右旋全方位移动；
2. 升降控制：可调节小车承载平台上升、下降，适配床体对接高度；
3. 紧急停止：点击停止按钮，小车立即终止所有运动；
4. 自动寻位：点击「床机自动寻位」，小车自动精准对接手术床，无需人工操控。
3.5.3 速度调节
支持独立设置移动、旋转、升降、自动寻位的运行速度，可通过滑块或数值输入精准调整，适配不同操作精度需求。
四、右侧辅助边栏功能说明
4.1 面板基础属性
右侧边栏与左侧控制面板对称布局，视觉风格统一，采用磨砂半透明设计，从上至下分为手术进度、场景预设、设备辅助功能三大模块，全程辅助手术流程管控，无需切换页面即可完成各类辅助操作。
4.2 核心功能模块详解
4.2.1 手术进度卡片
覆盖手术全流程管控，分为术前准备、体位调整、手术操作、术后复位四大阶段，实时展示当前阶段、进度占比、剩余操作时长及阶段操作注意事项。支持手动切换上下阶段，切换后设备自动适配对应初始姿态，规范手术操作流程。
4.2.2 场景预设卡片
内置四套专属手术室场景方案，一键切换设备整体布局，适配不同手术类型：
常规手术场景：默认场景，设备归位待机，适配普通外科手术；
影像手术场景：自动校准C臂初始姿态，开启影像预备模式，适配造影、透视手术；
微创手术场景：优化设备布局，预留充足微创操作空间；
术后复位场景：一键复位所有设备至待机状态，方便术后消杀、设备整理。
场景切换全程平滑过渡，切换过程自动锁定设备操作，避免指令冲突，保障设备安全。
4.2.3 辅助功能卡片
防碰撞保护：默认永久开启，系统实时检测设备间距，自动规避设备碰撞风险，全程保障手术设备安全运行。
束光器功能：开启后可可视化编辑影像照射光束范围，手动调整光束大小、角度，精准定位透视区域，支持参数保存复用。
4.3 通用操作规范
1. 术前优先匹配对应手术场景，完成设备初始布局校准；
2. 术中通过进度卡片把控手术节奏，严格按照阶段规范操作；
3. 所有功能卡片支持折叠、展开，可自定义界面展示样式，适配个人操作习惯。
五、中央3D可视化场景操作
5.1 区域核心定位
中央3D场景为系统核心可视化区域，1:1实景还原手术室手术床、C臂、AGV小车所有设备形态，设备姿态、位置变化实时同步更新，实现虚实完全同步，操作人员可直观查看设备实时状态。
5.2 基础视角交互操作
操作动作	操作方式	操作效果
视角旋转	鼠标左键按住拖拽	360°全方位旋转场景，无死角查看设备姿态
场景平移	鼠标右键按住拖拽	整体平移场景，调整设备可视位置
场景缩放	鼠标滚轮滚动	放大查看设备细节、缩小预览全局布局

5.3 悬浮快捷控制按钮
场景内置常驻悬浮功能按钮，优先级高于普通视角操作，规避手术误操作：
视角锁定/解锁：锁定后禁用所有鼠标视角操作，防止术中误拖拽、误缩放视角，手术过程建议全程锁定；解锁后可自由调整视角。
一键复位：重置手术床位置。
5.4 设备联动悬浮卡片
场景实时展示AGV小车、C臂设备状态卡片，直观显示设备运行状态、关键参数，点击卡片「更多」按钮，可快速跳转至对应设备控制面板，无需手动切换页面，提升操作效率。
5.5 束光器可视化功能
专为影像手术设计，实现透视光束可视化管控：
1. 开启束光器模式后，3D场景实时渲染影像照射光束范围，直观展示透视区域；
2. 支持手动拖拽光束四个顶点，自定义调整光束大小、角度、照射范围；
3. 调整后的光束参数可实时保存，支持一键复用，适配重复性手术场景。
六、底部全局功能栏操作
6.1 基础布局说明
底部功能栏为系统全局常驻工具栏，布局规整、功能优先级明确，涵盖应急安全、流程管控、系统设置、日志溯源、远程协作核心功能，适配术前、术中、术后全流程使用。
6.2 核心功能逐一说明
6.2.1 紧急制动（最高安全优先级）
红色高亮专属应急按钮，为系统最高权限安全功能，用于突发设备异常、姿态错位、存在碰撞风险等紧急场景。
触发效果：一键停止手术床、C臂、AGV小车所有设备运动，锁定全部设备操作权限，杜绝安全隐患。
解锁方式：再次点击按钮，即可解除锁定，恢复设备正常操控。
6.2.2 阶段切换
联动右侧手术进度卡片，快速切换手术全流程阶段，切换后系统自动适配各阶段设备预设姿态，简化人工调试步骤，规范手术流程。
6.2.3 影像重置
专属C臂设备快速复位功能，点击后一键清空所有自定义角度、偏移参数，将C臂所有关节恢复至出厂初始姿态，快速重置设备状态。
6.2.4 系统设置
进入全局系统配置页面，支持设备参数校准、界面样式调整、网络连接配置、操作权限管控、日志存储规则设置等基础配置。
6.2.5 操作日志
全程自动记录所有设备操作、参数修改、模式切换、异常状态信息，包含操作时间、操作内容、设备状态，支持实时查询、导出留存，完全满足医疗操作溯源规范。
6.2.6 数字协作
多人远程协同工具，支持多科室医护远程联动，包含高清视频连线、屏幕共享、实时消息通讯功能，适配远程会诊、异地手术指导、多团队协同手术场景。
七、系统通用规则与安全机制
7.1 多设备智能联动规则
系统自动实现手术床、C臂、AGV小车协同运行，规避设备冲突：
1. 手术床姿态变动时，C臂根据当前联动模式自动适配角度与位置；
2. 多设备同时运行时，系统自动优先级排序，紧急制动、体位切换指令优先级最高；
3. 所有设备状态实时同步界面，保证操作与设备实际状态完全一致。
7.2 标准化状态提示机制
所有操作、设备状态变更均弹出标准化提示，快速判断操作结果：
绿色提示：操作成功，包含体位应用、模式切换、设备复位等操作；
黄色提示：状态预警，包含设备低电量、参数临近限值、信号偏弱等；
红色提示：操作失败或设备异常，包含断连、指令失效、设备故障等；
灰色提示：常规状态变更，包含设备待机、视角重置、模式切换提醒等。
7.3 网络自动重连机制
1. 系统实时监测设备网络连接状态，通过顶部网络图标直观展示；
2. 网络断开后，系统自动启动多次重连机制，重连成功自动恢复设备状态；
3. 重连失败自动锁定高危设备操作，弹出异常提示并记录日志，保障设备安全。
7.4 智能操作锁定机制
系统根据设备状态自动管控操作权限，从源头规避误操作：
1. 设备运行过程中，禁止下发冲突指令、禁止切换核心联动模式；
2. 紧急制动触发后，全设备操作锁定，解锁后方可继续操作；
3. 特殊手术体位下，自动锁定高危设备移动功能，防止设备碰撞。
八、标准手术操作流程规范
8.1 术前准备流程
1. 设备自检：开机后查看顶部蓝牙、网络、信号状态，确认所有设备正常连接、无异常告警；
2. 场景适配：根据本次手术类型，在右侧边栏选择对应场景预设，完成设备初始布局校准；
3. 参数校准：按需调整设备运行速度、C臂联动模式、AGV运行参数；
4. 体位预演：选定手术所需标准体位，3D场景预览姿态，确认无设备干涉、适配手术需求。
8.2 术中常规操作流程
1. 通过左侧控制面板精准调节手术床、C臂、AGV设备姿态，适配手术操作；
2. 依托中央3D场景实时观察设备状态，通过右侧进度卡片把控手术流程节奏；
3. 按需开启束光器可视化、防碰撞保护功能，切换设备联动模式；
4. 遇设备卡顿、姿态异常等突发情况，立即触发紧急制动，排查故障后恢复操作。
8.3 术后收尾流程
1. 切换至术后复位场景，一键将所有设备复位至待机初始姿态；
2. 清空所有自定义体位、设备参数、临时配置，恢复系统默认状态；
3. 查看并导出本次手术操作日志，完成操作溯源留存；
4. 关闭设备联动、随动功能，将设备切换至待机休眠状态，完成术后整理。

十、常见故障排查说明
10.1 界面显示异常
场景模型不刷新：排查设备网络连接状态，刷新界面同步设备最新姿态，重启场景渲染；
面板数据卡顿：退出速度设置等编辑模式，重新切换设备页面，同步后端数据；
功能按钮无响应：检查设备是否处于运动锁定、紧急制动锁定状态，解锁后即可恢复操作。
10.2 设备控制异常
设备无法运动：确认网络正常、无紧急制动锁定、设备参数在合法范围，解除锁定后重试；
C臂随动失效：重新切换联动模式，检查手术床运行状态，重启设备联动机制；
AGV自动寻位失败：检查设备电量、信号强度，清理周边障碍物，重新发起寻位指令。
10.3 网络连接异常
网络图标红色断连：检查设备网线、无线连接状态，确认后端服务正常，手动触发系统重连；
数据延迟过高：清理系统冗余进程，优化网络带宽，重新建立设备通信连接。
